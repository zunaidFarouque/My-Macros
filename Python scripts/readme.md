# Python Macros Workspace

This workspace lets you easily set up one-click shortcuts for any Python script.
Just follow these steps:

1. **Add Scripts:**  
   Create subfolders here and place your Python scripts inside them.

2. **Generate Shortcuts:**  
   Run `Shortcuts creator.py`. That’s it!

Your ready-to-use shortcuts will appear in the `Shortcuts` folder.
No compiling or extra setup required.
